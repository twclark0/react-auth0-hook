# Stacks

- pop(): Remove top item from stack
- push(i): Add an item to top of stack
- peek(): Return the top of stack
- isEmpty(): Returns true if stack is empty



---

<CodeSurferLayout>

```js  title="Stacks"

  this.stack = []
  this.pop = x => {}
  this.push = x => {}
  this.peek = () => {}

}

const myStack = new CreateStack()
```
